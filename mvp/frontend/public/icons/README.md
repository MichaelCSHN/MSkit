# 底图工具栏图标

把你提供的三张 PNG 保存到本目录，文件名严格如下（小写）：

| 文件名 | 用途 | 对应图片 |
|---|---|---|
| `street.png` | 街道底图按钮 | 蓝色小球 + 街道地图 |
| `satellite.png` | 卫星影像按钮 | 地球（globe） |
| `terrain.png` | 3D 地形按钮 | 木质分层等高线地形 |
| `locate.png` | 📍设为起点按钮 | 航拍街区 + 红色定位针 |

- 建议为方形（如 256×256 或 512×512），会被裁成圆角小图标。
- 缺文件时按钮自动回退到 emoji（🗺️ / 🛰️ / ⛰️ / 📍），不会报错。
- 保存后硬刷新（Ctrl+F5）即可。
