# MSkit v1.3.2 BOM 成本同步表

> **基准**：小批量 50 套（MOQ50）｜**状态**：内部评审稿｜**用途**：同步主规格 v1.3.2 的地区化 SKU、电池方案与 Pilot Batch 经济性。历史明细见 `MSkit_v1.3_BOM_Cost_Table.md`。

---

## 1. 成本口径

| 口径 | 说明 |
|------|------|
| 基础 BOM | 仅硬件、结构、线材、组装与基础测试，不含研发摊销、认证摊销、渠道、售后、物流、关税、坏件损耗 |
| Pilot Batch | 首批 50 套定义为试点批次，不承担全部研发回收 |
| 研发摊销 | 商业模型建议按 300–500 套摊销，或对 Pilot Batch 单独定价 |
| 软件/授权 | YOLOv8/Ultralytics 商用授权、数据标注、SCA/SBOM 工具不计入基础 BOM，应作为年度软件成本或 COGS 单列 |

## 3. 通信地区化 SKU 修订

| 部件 | v1.3 历史写法 | v1.3.2 执行口径 | 说明 |
|------|---------------|-----------------|------|
| LoRa 模块 | Ebyte E22-900T30D | Ebyte E22 系列，按 CN/US/EU 锁定频段 | 不再写“868/915MHz 通用” |
| LoRa 天线 | 868/915MHz 天线 | CN / US / EU 对应频段天线 | 与模块频段、法规 duty cycle 匹配 |
| LTE 模块 | Quectel EC25-EU | EC25-EU / EC25-AF / 中国合规模块等，按市场选型 | 避免单一 EU 型号被误用于所有市场 |
| Wi-Fi 6E | Intel AX210 | 按目标市场确认 6GHz 频段开放状态 | 中国、美国、欧盟分别确认 |
| 卫星通信 | Iridium 9603N 可选 | 仅作为 SOS 备份 SKU；地区合规另审 | 不作为基础 BOM 默认项 |

## 4. 电池方案修订

| SKU 层级 | 推荐化学体系 | 单块能量 | 运输口径 | 备注 |
|----------|--------------|----------|----------|------|
| 标准民用版 | Li-ion 智能电池包或 LiFePO4 | 目标 <300Wh 优先 | 尽量降低危险品运输复杂度 | 需重新核算续航与结构空间 |
| 专业/工程样机 | 6S LiPo | Backpack 355Wh / Team 488Wh | >300Wh 默认按危险品运输管理 | 仅用于工程样机或专业用户版本 |
| 外接扩展电池 | LiFePO4 优先 | 视客户场景 | 按目标市场运输/仓储/保险流程确认 | 不作为基础版默认配置 |

> UN38.3 仅说明通过运输测试，不等于可随意空运、随身携带或普通快递运输。销售前必须确认目标市场的空运、陆运、仓储、保险和售后退换货流程。

| 变体 | 当前零售价 | 若含 $8k 研发摊销 + 20% 渠道 + 5% 售后，建议最低价 |
|------|-----------|--------------------------------------------------|
| Backpack Base | $7,999 | ≥ $13,000–$14,500 |
| Backpack Thermal | $11,999 | ≥ $18,500–$20,500 |
| Team Base | $14,999 | ≥ $14,500–$16,500 |
| Team Thermal | $17,999 | ≥ $20,000–$22,500 |

**结论**：首批 50 套应明确为 Pilot Batch。量产商业模型应以 300–500 套摊销周期、地区化 SKU 和软件授权成本重新建模。
