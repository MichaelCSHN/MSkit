## 四、各变体BOM总览

| 变体 | 基础BOM | +热成像 | 目标零售价 | 毛利率(估算) |
|------|---------|---------|-----------|-------------|
| Backpack Base | $1,844 | - | $7,999 | ~77% |
| Backpack + Thermal | $1,844 + $4,108 = $5,952 | 含 | $11,999 | ~50% |
| Command Base | $2,680 | - | $14,999 | ~82% |
| Command + Thermal | $2,680 + $4,108 = $6,788 | 含 | $17,999 | ~62% |

> **注**：目标零售价含研发分摊、认证费用、售后预留、渠道利润。热成像模块占Thermal版BOM成本约69%，是主要成本驱动因素。
> **v1.3 校对订正**：基础BOM由 $1,270 订正为 $1,844，Command 增量由 $1,494 订正为真实差额 $836（Command Base 由 $2,764 变为 $2,680）。相应毛利率由 84%/55%/82%/62% 调整为 77%/50%/82%/62%。Command 版因两处误差方向相反而基本抵消，毛利率几乎不变；Backpack 版毛利率下降约 7 个百分点。
