## 1. 项目概述

### 1.1 目标场景（纯民用/商用）

| 场景 | 核心需求 | 合规边界 | 示例用例 | 优先级 |
|------|----------|----------|----------|--------|
| Milsim（军迷模拟游戏、安保培训、Airsoft/Paintball 大型活动） | 红/蓝方态势模拟、行动推演、实时通信 | 严禁军用术语；宣传使用"民用模拟""安全作业"表述 | 10–50 人团队演练，Mesh 支持 8 节点实时位置共享 | P1 |
| 狩猎与野外追踪 | 动物热源探测、轨迹追踪、区域监控 | 遵守野生动物保护法；禁止非法捕猎 | 热成像 + Trailcam 联动，自动生成每日活动报告 | P2 |
| 野外作业安保与巡检 | 周界监控、人员定位、异常告警 | 隐私保护；监控区域设置明显标识 | 工业园区/林区巡检，AI 异常检测触发语音/LED 告警 | P2 |
| 救援演练与灾害响应 | 人员搜救、热源定位、路径规划、资源调度 | 与应急部门协调；数据脱敏 | 地震/洪水模拟，离线路径优化 + 一键 SOS | P1 |
| 户外探险与团队协调 | 团队定位、地形导航、安全预警 | 用户知情同意；位置数据本地优先 | 登山/徒步团队，离线地图 + 落单检测 | P3 |

### 1.2 设计目标

MSkit 旨在提供一台便携、加固的边缘信息中心，支持离线优先的多源感知、AI 辅助决策与任务管理。系统严格遵循民用法规，所有功能设计均围绕民用场景展开，杜绝任何军事化表述或功能。核心设计原则包括：

- **合规优先**：所有功能通过法律审查
- **安全默认**：数据加密、访问控制、审计日志
- **实用导向**：解决真实痛点，非炫技
- **可扩展架构**：模块化软硬件，支持后续升级
- **成本可控**：小批量 BOM < $6,000（基础版）
- **可维护性**：FRU 现场可更换，MTTR < 30 分钟

| 属性 | Backpack 版 | Command 版 |
|------|-------------|------------|
| 电池 | 6S 16000mAh LiPo ×2（热插拔） | 6S 22000mAh LiPo ×2（热插拔） |

> **电池选型说明**：采用航模级 LiPo 电池（Herewin/Gens ace），已通过 UN38.3 认证[^42]。6S 16000mAh 单价约 $115/块。热插拔设计允许在不关机情况下更换电池。MTBF 按 500 次循环计算，每块电池寿命约 2–3 年（视使用频率）。
> **安全警告**：LiPo 电池需严格 BMS 保护，禁止过放（< 3.0V/节）和高温（> 60°C）充电。

**热管理方案：**

- 主动散热：Jetson 专用涡轮风扇（PWM 调速，40dB@全速）
- 被动散热：铜管均热板 + 铝制散热鳍片
- 温度监控：CPU/GPU/电池/外壳各布 1 路 NTC 温度传感器
- 过热保护：> 85°C 自动降频，> 95°C 强制关机
- 低温预热：-20°C 环境下启动前电池预热（PTC 加热片，5W）

| 编排 | systemd（服务管理）/ 可选 K3s | - | 进程/服务管理 |
| AI 框架 | PyTorch 2.2 + TensorRT + ONNX Runtime | 容器化部署 | 模型推理 |

> **JetPack 版本锁定说明**：JetPack 6.2.1 为 2025年7月发布的生产稳定版，支持所有 Jetson Orin 模块，基于 Ubuntu 22.04 LTS[^81]。关键修复包括：Docker 28.x 兼容性、HSM 签名支持、刷写成功率提升。不建议使用 JetPack 6.1（存在 SSD 刷写失败问题[^80]）。NVIDIA 已将 Jetson Orin 产品生命周期延长至 2032 年[^35]。

**最高优先级风险：**
